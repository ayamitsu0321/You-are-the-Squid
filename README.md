## Features
- Players be a squid!
- Player can breath underwater.
- Player swim fast underwater.
- Player dig fast underwater.

## Requirement
- Minecrat 1.15.2
- MinecraftForge 31.1.0+

## Installtion
- Install MinecraftForge.
- Puts .jar file into mods folder.
